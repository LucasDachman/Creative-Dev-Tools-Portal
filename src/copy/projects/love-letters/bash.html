Sorry Bash, there's someone else.

You were a great companion to my command line and your control over the OS is undeniable. But it's time to let you go. There's someone else and her name is zsh. Zsh has some great features like automatic cd, better code completion, spelling correction and tons of custom plugins.

Some awesome bash (and zsh) commands

cd
change directories
mkdir
create a new directory
touch
create a new file
rm
remove a file (use -r to remove a directory)
mv
rename or move a file or directory
ls
list the contents of a directory (-la for extra information and hidden files)
cat
output the contents of a file
man
the manual
chmod
edit the permissions of a file
